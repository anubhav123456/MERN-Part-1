# Node.js `os` Module

## 1. What is the `os` module in Node.js?

**Answer:**

The `os` module is a built-in Node.js module that provides information about the operating system.

It is commonly used to get:

* Operating system details
* CPU information
* Memory information
* Home directory
* Temporary directory
* System architecture

---

## 2. Do we need to install the `os` module?

**Answer:**

No.

`os` is a built-in Node.js module, so it can be imported directly.

```typescript
import * as os from "node:os";
```
